Customize Ranking: Popularity Java API and Library

Instructions:
  • Select the Table Type:
    • APIs: Occurrence of each Java API. Enter the list separated by comma (example: java.util.List, java.util.ArrayList).
    • Libraries: Occurrence of library's APIs. Enter the libraries list separated by comma (example: java.util, java.io).
      • Group Libraries: Mark option to group by libraries.
  • Then, press the Research button.
  • Press the Delete table button to remove data table.
  • Press the Download button to download data in CSV format.
