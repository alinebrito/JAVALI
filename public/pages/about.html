JAVALI - JAVA Libraries and Interfaces

Everyday, libraries/frameworks are updated, created or removed, and so their APIs. Such changes may impact stakeholders such as APIs providers and clients. In this context, it is important for APIs providers to know strategic information like APIs change impact on clients and popularity over competitors. On the client side, it is interesting to compare APIs in order to select the most recommended in the market. To address these challenges, we propose JAVALI, a tool that allows analize popularity of the Java Interfaces and Libraries.
