Project JAVALI

JAVALI (Java Library and Interface) is a tools to analize info about popularity Java API's. It is a project developed for the discipline Maintenance and Software Evolution, of the Master's at Computer Science from Federal University of Minas Gerais (UFMG).

Data were collected through the Boa Language and Infrastructure. We create a Boa script for collect info about use Java API.

We database contains:
  • 263.425 Java projects;
  • 16.386.193 Java files;
  • 131.147.733 Java imports;
  • 4.780.469 imports distincts;
  • 3.803.878 imports belong to a single project.

We use the GitHub database's of Boa Project . We use projects with valid Java files and at least one import.
The JAVALI design was inspired by apiwave developed by André Hora. It allows to analyze on the most used APIs in top 500 GitHub java projects.
